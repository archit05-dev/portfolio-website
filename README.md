# Personal Portfolio Website

A personal portfolio website built with HTML and CSS to showcase my projects, technical skills, and professional profiles.

## Live Demo

[View Portfolio](https://archit05-dev.github.io/portfolio-website/)

## Features

* About Me section
* Skills section
* Project showcase
* GitHub and LinkedIn links
* Responsive design for desktop and mobile devices

## Technologies Used

* HTML5
* CSS3
* Flexbox
* Media Queries

## Featured Project

### Library Management System

A Java-based console application that supports:

* Book management
* User management
* Issue and return workflows
* Transaction tracking

## Run Locally

1. Clone the repository:

```bash
git clone https://github.com/archit05-dev/portfolio-website.git
```

2. Navigate to the project folder:

```bash
cd portfolio-website
```

3. Open `index.html` in your browser.
